## Usage

Skills are stored in `~/.claude/skills/` and auto-loaded by Claude Code.

## Setup

```bash
git clone git@github.com:navigotechsolutions-labs/navigo-claude-skills.git
cp -r navigo-claude-skills/* ~/.claude/skills/
```
