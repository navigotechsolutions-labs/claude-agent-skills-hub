# Summary Templates

## Executive Summary Template

```
## Executive Summary

**Recommendation:** [One-sentence recommendation or key takeaway]

**Key Metrics:**
- [Metric 1]: [Value] ([change from baseline])
- [Metric 2]: [Value] ([change from baseline])

**Background:** [1-2 sentences on context]

**Key Findings:**
1. [Most important finding with specific number]
2. [Second finding with specific number]
3. [Third finding with specific number]

**Risks:**
- [Primary risk and mitigation]
- [Secondary risk]

**Next Steps:**
- [ ] [Action item] — [Owner] — [Deadline]
- [ ] [Action item] — [Owner] — [Deadline]
```

## Technical Summary Template

```
## Technical Summary

**Purpose:** [What this component/system/document addresses]

**Approach:** [High-level description of the solution in 2-3 sentences]

**Key Decisions:**
- [Decision 1]: [Choice made] because [rationale]
- [Decision 2]: [Choice made] because [rationale]

**Dependencies:**
- Requires: [List of dependencies]
- Required by: [List of dependents]

**Trade-offs:**
- Gained: [What was optimized for]
- Sacrificed: [What was deprioritized]

**Limitations:**
- [Known limitation 1]
- [Known limitation 2]

**Open Questions:**
- [Unresolved question 1]
- [Unresolved question 2]
```

```
### Decisions Made

- [ ] [Task] — [Owner] — due [Date]
- [ ] [Task] — [Owner] — due [Date]

- **[Topic 2]:** [Summary of discussion and outcome]

### Open Questions

```

## Research Paper Summary Template

```
## Paper Summary: [Title]

**Authors:** [Names] | **Published:** [Date] | **Venue:** [Journal/Conference]

**Research Question:** [What was investigated]

**Methodology:** [Brief description of approach, sample size, duration]

**Key Findings:**
1. [Finding with specific data point]
2. [Finding with specific data point]
3. [Finding with specific data point]

**Significance:** [Why these findings matter]

**Limitations:**
- [Limitation of methodology]
- [Limitation of scope]

**Implications:** [What should change based on this]
```
