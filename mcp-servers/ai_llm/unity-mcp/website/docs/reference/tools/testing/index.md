# `testing` tools

Test runner & async test jobs

- **[`get_test_job`](./get_test_job.md)** — Polls an async Unity test job by job_id.
- **[`run_tests`](./run_tests.md)** — Starts a Unity test run asynchronously and returns a job_id immediately.
